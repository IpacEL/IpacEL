```
功能特性等: 
	- 免安装: 可以在没有安装 Java 的系统里启动 MC
	- 性能优化: 使用新的软件和优化模组, 预设配置等
	- 清理日志: 在每一次正常关闭启动器后 [自行添加的功能产生的文件可能不在清理范围内]
	
	#其他
	- 如果MC帧数低可以尝试将 最大帧率改为垂直同步, 按下F11进入全屏
	- 如果您不希望启动器在后台运行, 只需要在HMCL游戏设置中将 启动器可见性 设置为 游戏结束后结束启动器
		这样做可以让Ipac和HMCL不在后台等待MC运行结束, 但会失去文件清理和错误显示等功能
	- HMCL会在 %appdata% 目录下生成 .html 目录来存放一部分配置和日志[可能还有JFX]
		在14.7版本中已移除了对这个目录中不属于日志文件的清理, 需要删除的用户可在资源管理器输入 %appdata% 手动删除
	- 启动器背景图片和游戏列表中的游戏版本图标来自Ipacamod
```
