核心会自动删除配置中的注释, 可以将带有注释的配置保存到其他位置. 编辑时修改带有注释的那份, 应用时覆盖到服务器目录里.  
工具 "YAML配置注释同步", 用于更新配置时同步旧配置中的注释: [/web/yaml_json_1/].  
所有配置注释均由自己查文档翻译[少量机翻], 已修改的配置是自己服务器使用的值[或是推荐使用的].  
部分注释后面带问号表示不确定翻译是否正确[配置比较多, 一部分还没有检查...]部分注释后面的值表示默认值.
